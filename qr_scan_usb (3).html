สแกน QR‑Code ด้วยกล้องมือถือ

ต้องเปิดผ่าน HTTPS หรือ http://localhost
ยังไม่ได้เริ่มสแกน
เคล็ดลับ: ถ้าอุปกรณ์มีหลายกล้อง ให้ลองเลือกกล้อง "Back"/"Wide" เพื่อผลลัพธ์ดีที่สุด
ผลที่อ่านได้ (กดคัดลอกได้)
แสดงเป็น JSON (ถ้าข้อความเป็น JSON) 
(รอข้อมูล)

หากเปิดหน้านี้ใน <iframe> หรือป๊อปอัป จะส่งค่าที่อ่านได้ให้หน้าแม่อัตโนมัติผ่าน postMessage (ชนิด qr-decoded) 

// หน้าแม่
window.addEventListener('message', (ev)=>{
  if(ev.data && ev.data.type==='qr-decoded'){
    console.log('ได้ค่า:', ev.data.text);
  }
});
